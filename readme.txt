== Description ==
'Nusratech Pricing Block', A Gutenberg block to create 3 column maximum pricing table in a row.

== Installation ==

1. Upload the `nusratech-pricing-block` folder to your `/wp-content/plugins/` directory or alternatively upload thenusratech-pricing-block.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your local computer.
2. Install and activate the Gutenberg WordPress plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add pricing tables within Gutenberg with the new "Pricing Block" block.
